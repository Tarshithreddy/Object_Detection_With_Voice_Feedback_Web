🚀 Live Demo
👉 https://object-detection-with-voice-feedback-web.onrender.com/

📌 About the Project
This is a real-time object detection system using YOLOv4-Tiny, Flask, and OpenCV.  
It detects objects from webcam input and gives voice feedback for assistive use.

⚙️ Features
- Real-time object detection
- Voice feedback using browser speech API
- Object position detection (left / center / right)
- Distance estimation (approximate)

🧠 Tech Stack
- Python
- Flask
- OpenCV
- YOLOv4-Tiny
- JavaScript (frontend)

▶️ How to Run Locally
```bash
pip install -r requirements.txt
python app.py
